Ключевые этапы загрузки:
- **UEFI/BIOS**: Начальная инициализация. Запуск прошивки, проверка железа, запуск загрузчика;
- **Boot-loader**: Загрузчик запускает ядро kernel.
- **Kernel**: Ядро, которое монтирует FS и запускает процессы.
- **init/systemd**: [PID](Процесс) 1, который управляет всеми сервисами(процессами).

Загрузка системы - это линейный процесс, но с ветвями (например, initramfs для ранней инициализации).
