**SUDO** (Super User Do) - то команда, позволяющая выполнить инструкцию в терминале от имени [root пользователя (UID = 0)](Пользователи).

Командой `sudo` может пользоваться только тот *пользователь, который входит в группу sudo.* При установки Ubuntu, первый пользователь автоматически добавляется в эту группу.

sudo - это компромисс использования root пользователя. При работе под root, ядро видит [UID 0](Пользователи), а этому юзеру ядро не отказывает не в чём. *sudo позволяет выполнять определенные команды с root правами, и самое главное - это логируется.*

Правила, описывающие кто и что может делать под SUDO описаны в файле `/etc/sudoers`
