`usermod [options] LOGIN `# Команда позволяет модифицировать юзера

`usermod -a` -a append, добавляет пользователя в еще одну группу. Или точнее, добовляет пользователю дополнительную группу(ы). Используется только с -G

`usermod -G `# -G groups GROUP1[,GROUP2,GROUP3,...[GROUPN]]. Эта опция позволяет задать пользователю несколько групп, через запятую, без пробелов. 
!!! При этом, если пользователь входит в какие-то группы, которые не будут перечислены при использовании этой опции, он будет удалён из этой группы.
НО. Можно использовать опцию `-G` с опцией `-a`. Тогда, `-aG` будет работать как -a, но для перечисленных групп.

###  Управление пользователями
`useradd -m -s /bin/bash username`
Добавляет пользователя. Использование этих двух флагов `-m`, `-s` считается хорошей практикой.

`-m, --create-home` - создать пользовательскую директорию, если не существует. Файлы и папки, содержащиеся в skeleton directory (/etc/skel), будут скопированы в /home директорию. Skeleton dir. can be defined with the -k option.

`-s, --shell SHELL` Устанавливает путь к [оболочки входа, см. */nologin](Пользователи) (login shell)

##### su
```bash
su [option] [-] [user [argument]]
```
Команда **su** - это команда переключения на другого пользователя. `su - [user_name]`. При переключении меняется: окружение, доступы, домашняя директория того пользователя на которого ты переключаешься. Так остаётся пока не напишешь `exit`

```bash
su - anna # Стал Анной. Всё что делаю дальше — от её имени
exit # Вернулся к себе
```

Если `su` используется без передачи аргумента, то ты автоматически переключаешься на **root**. `sudo su`

### chown 
`chown` - команда для установки роли на файл.

Пример:
```bash
sudo chown dispetcher:dispetcher /home/dispt/chromExt

# формат chown пользователь:группа
```

`chown -R` - для рекурсивного изменения роли.

### chmod
```bash
chmod 755 file # числовая запись
chmod u+x file # симвользная: добавить execute владельцу
chmod go-w file # убрать write у группы и остальных
chmod -R 755 /dir # рекурсивно изменить права для директории
```
