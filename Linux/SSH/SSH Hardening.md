Я SSH Hardening (упрочнение\закалка SSH) - это процесс конфигурации службы для максимального повышения безопасности удаленного доступа.

SSH полностью настраивается в `/etc/ssh/sshd_config`

Основные правила sshd:
1. Отключить авторизацию под root через SSH
2. Отключить аутентификацию по ssh через пароль.
