Пакеты - это архив с готовым(скомпилированным) ПО, библиотеками, конфигами, [бинарниками](Бинарный%20файл.md), документацией, необходимыми для установки ПО. 

В debian-bas (Ubuntu, Debian) - формат пакетов .deb.

Пакеты упрощают установку: вместо компиляции из исходников, просто скачиваешь и ставишь.

**Ключевые особенности пакетов:**
- **Состав:** Включают [бинарные файлы](Бинарный%20файл.md) (исполняемые файлы), документацию, скрипты установки/удаления и метаданные (версия, описание, зависимости).
- [Зависимости](Зависимости): Пакеты "знают", какие другие компоненты нужны для их работы, что предотвращает конфликты версий.
- **Управление:** Вместо поиска файлов вручную, менеджеры пакетов (например, `apt`, `yum`) устанавливают ПО из [репозиториев](Репозитории).
- **Форматы:**
    - `.deb`: Используются в Debian, Ubuntu, Linux Mint.
    - `.rpm`: Используются в Red Hat, Fedora, CentOS, ALT Linux.
- **Хранение:** Установленные пакеты часто кэшируются в `/var/cache/apt/archives` (для deb-систем).
