Зависимости в Linux - это внешние программные компоненты, библиотеки, пакеты, необходимые конкретной программе для корректной установки и работы.

Обычно зависимости - это файлы библиотек (`.so`), интерпретаторы, драйверы или другие приложения, которые требуются но не включены в основной исполняемый файл программы.

[Пакетные менеджеры](Пакетные%20менеджеры) в Linux обычно автоматически находят и устанавливают все необходимые зависимости при установки программы, что сделано чтобы избежать dependency hell - конфликтов версий.
