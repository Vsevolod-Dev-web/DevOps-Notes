Пакетные менеджеры в Linux - это инструмент для автоматической установки, обновления и удаления ПО ([пакетов](Пакеты)), которые обеспечивают целостность системы и управляют [зависимостями](Зависимости). 

Они находят нужный софт в [репозиториях](Репозитории), скачивают его и устанавливают, заменяя ручную компиляцию и установку файлов.

Для Debian: APT (высокий уровень, frontend) + dpkg (низкий уровень, работает с .deb напрямую). APT умный: решает зависимости, интерфейс с репозиториями. 

- APT (команды: apt, apt-get, apt-cache) - удобный, handless зависимости.
- dpkg - базовый, без авто-зависимостей, но полезен для локальных .deb
