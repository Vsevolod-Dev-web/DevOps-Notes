Приоритет проесса - инструмент позволяющий управлять тем, какой процесс будет выполняться раньше\дольше других.

#### Приорететы:
Планировщик процессов **SCHED_OTHER**, стараеться быть "абсолютно справедливым". Это стоит понимать так: планировщик старается дать всем процессам поровну времени процессора - быть "справедливым". Влиять на это можно с помощью механизма "вежливости" - **Nice-Value**.
- **Nice value** - обычные приоритеты (-20 до 19):
	- -20 - наивысший приоритет (только root). Самый "НЕвежлевый"
	- 0 - приоритет по умолчанию;
	-  19 - наименьший приоритет. Самый вежливый

Есть задачи, которым нужна гарантированная реакция (аудио\видео стриминг, управление станки). Nice-value тут недостаточно. Тут работает механизм **Real-Time** (RT). Для этого существует отдельный планировщик задач, игнорирующий "Nice-Value". Он работает с приоритетами реального времени.
- **Real-time** приоритеты - для критически важных задач (1-99):
	  - **SCHED_FIFO** - процесс с одинаковым приоритетом должны работать в строгой очереди.
	  - **SCHED_RR** (Round Robin) - Похож на FIFO, но процессам с одинаковым приоритетом выделяются фиксированные [кванты времени](Квант%20времени.md) (time slices). Если процесс не выполнился за отведённый квант времени, он переходит в конец очереди.


#### Практика

Для работы с обычными процессами исп. две команды: `nice` и `renice`

```bash
# Запустить с nice-значением 15 (низкий приоритет)
nice -n 15 ./backup.sh

# Если нужно запустить с высоким приоритетом (отрицательное nice), нужны права root
sudo nice -n -10 ./critical_app
```

Изменить приоритет работающего процесса:

```bash
ps aux | grep backup.sh
# или
pgrep -a backup.sh

# Понизить приоритет процесса (сделать его "вежливее")
renice 15 -p 1234
# Повысить приоритет (сделать его менее "вежливым") может только root
sudo renice -5 -p 1234
```

#### Резюмируя

NI (Nice Value) - значение, описывающее насколько процесс "жаден" до CPU. Устанавливаеться пользователем.
RPI - его фактический приоритет для ядра. Устанавливаеться планировщиком CFS (Complitly Fair Schudluler).
