Управление [процессами](Процесс.md) - это просмотр запущенных программ, их завершение, перевод в фон\передний план и контроль приоритетов.

#### Основные команды 
**Просмотр процессов:**
- **ps** **aux**: Просмотр процессов (статичный список) с детализацией.
- **ps -eo pid, ppid, comm, state**: PID, Родитель, команда, состояние;
- **top** или **htop** (если установлен): Интерактивный мониторинг (динамичный, с CPU, RAM).

**Мониторинг процессов:**
- `free -h`: Использование памяти;
- `lsof -p <PID>`: Открытые файлы процесса;
- `vmstat 1`: Статистика системы каждую секунду;

**Анализ [состояний](Состояния%20процессов):**
```bash
# Поиск процессов по состоянию
ps aux | awk '$8 ~ /^D/ {print}'    # Процессы в состоянии D
ps -eo pid,ppid,state,cmd | grep Z  # Zombie-процессы

# Детальная информация о процессе
ps -p <PID> -o pid,ppid,%cpu,%mem,etime,cmd
cat /proc/<PID>/status              # Детальный статус

# Дерево процессов
pstree -p                          # Иерархия процессов
ps -ef --forest                    # Альтернативный вариант

```

**Команды управления [приоритетами](Приоритеты%20и%20планирование):**

```bash
# Изменение nice value
nice -n 10 command              # Запуск с приоритетом 10
renice 5 <PID>                  # Изменить приоритет процесса
renice -5 -u username           # Изменить приоритет всех процессов пользователя

# Real-time приоритеты
chrt -f 50 command              # FIFO приоритет 50
chrt -r 30 command              # RR приоритет 30
chrt -p <PID>                   # Показать приоритет процесса
```

- **kill**: Завершение процесса по PID ([сигналы](Сигнал): -9 для force kill).
- **&**: Запуск в фоне.
- **jobs**: Список фоновых задач в текущей сессии.
- **fg** / **bg**: Перевод в foreground/background.
- **Ctrl+Z**: Приостановить foreground процесс.
- **nice** / **renice**: Установка приоритета (nice value от -20 high до 19 low).
