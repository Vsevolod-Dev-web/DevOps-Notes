Монтирование - процесс присоединения ФС к дереву директорий, чтобы к ней можно было обращаться. Без монтирования диск или раздел просто "висит" в /dev, но не доступен для чтения\записи через обычные пути.

Когда ты монтируешь ФС, оно "приклеивается" к **точке монтирования** ([mount point](Точка%20монтирования%20(Mount%20point))).

Пример, если монтировать раздел `/dev/sda1` в `/mnt/data`, то файлы с раздела появятся в точке монтирования (`/mnt/data`);

**Права**. Монтировать может только root (или через sudo (super user do)). Обычные пользователи - с помощью fstab или udisks.

Монтирование через команду - временное (до перезагрузки). Постоянное - через `/etc/fstab`

#### Для чего это?
- Позволяет динамически добавлять\удалять ФС без перезагрузки.
- Поддерживает разные типы ФС (локальные, сетевые как NFS, облачные).
- Для Docker volumes, K8s PV (Persistent Volumes), backup и scalling
