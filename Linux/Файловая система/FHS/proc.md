### Определение
`/proc` - виртуальная файловая система, содержащая всю информацию о процессах, и не только.

Главные директории в ней: 
```bash
/proc/$pidNum/exe # Симлинк, указывает где файл запуска процесса
/proc/$pidNum/fd  # Открытые файловые дескрипторы процесса
/proc/$pidNum/status # общая информация о процессе
```

Так же полезно:
```bash
/proc/meminfo    # информация о RAM и swap
/proc/cpuinfo    # информация о процессоре
/proc/uptime     # сколько времени система работает
```
