**inode** - это структура данных на диске, которая хранит всю информацию о файле. Каждый файл имеет свой уникальный номер **inode**.

Что хранится в **inode**:
-  Тип файла;
- Права доступа;
- Владелец (UID, GID);
- Размер файла;
- Временные метки (создан, изменён, последний доступ);
- Указатели на блоки данных на диске (где физически лежат данные);

```bash
ls -i # Выдает список файлов, с inode каждого
stat file # Показывает детальную инфу по файлу
```

Так же, inode хранит **указатели на блоки диска**, где лежат данные файла.

Само имя файла хранится отдельно в [каталоге](Директория.md).
