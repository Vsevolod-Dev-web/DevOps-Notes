**Файловая система** - это структура описывающая методы хранения и обработки информации на диске. 

Linux, на стадии установки предоставляет выбор файловой системы (далее, ФС). Linux поддерживает разные типы FS, такие как ext4 (стандартная), btrfs (с snapshots), xfs (для больших данных). 

ФС в Linux представлена в виде дерева. 

Диски [монтируются](Монтирование.md) в ==дерево== с помощью команды `mount`

Файл `/etc/fstab`описывает, что монтируется при запуске.

Все каталоги (диски, разделы, сетевые ресурсы) являются предками [корневого](Корень), монтируются внутрь этого каталога.
