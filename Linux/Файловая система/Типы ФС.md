Есть различные [файловые системы.](Файловая%20система.md) Каждая из них, оптимизирована для определенной задачи.

Существуют следующие типы ФС:

1. **ext4** - стандартная ФС для Linux. Надёжная, проверенная временем, подходит для большинства задач. Это стандартный выбор для сервера.
2. **xfs**- оптимизирована под **файлы большого объема и высокую нагрузку.** Часто используется для серверов с большим объемом данных. (Видео, бэкапы, базы данных).
3. **tmpfs** - монтируется прямо в RAM. Данные исчезают после перезагрузки. Используется для временных файлов - например [/tmp](Стандарт%20FHS), часто это tmpfs.
4. **overlay** - особая ФС, которая **накладывает** один слой файлов на другой. Основа Docker образов.

Команды:

`findmnt` - позволяет узнать файловую систему в директории.
