**Symlink** (символическая ссылка) - это [файл](Файл), **[inode](inode)** которого стандартная. Но в блоке данных, который указан в inode содержится ссылка на другой файл.
```
Директория: "shortcut.txt" → inode #99 
inode #99: метаданные...
Блок данных #99: "/home/vsevolod/data.txt" ← просто путь
```

Если удалить файл, на который указывает символьная ссылка, то такая ссылка станет "битой" (dangling symlink)
