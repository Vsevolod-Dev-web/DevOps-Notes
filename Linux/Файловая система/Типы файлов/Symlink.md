**Symlink** (символическая ссылка) - это [файл](Файл), в **[inode](inode)** которого не содержится указание на блок данных (block pointer) в памяти. В место этого, там указана ссылка на другой файл.
```
Директория: "shortcut.txt" → inode #99 
inode #99: метаданные...
Блок данных #99: "/home/vsevolod/data.txt" ← просто путь
```
