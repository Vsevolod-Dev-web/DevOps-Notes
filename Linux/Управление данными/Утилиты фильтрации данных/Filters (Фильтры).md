Фильтры - это специальные команды bash, которые принимают данные на стандартный вход (stdin) и выдают стандартный вывод (stdout) и позволяют автоматизировать обработку больших объемов данных без создания временных файлов.

Фильтры позволяют производить следующие манипуляции с данными: сортировка, подсчет, форматирование и т.д.

Часто фильтры используются в цепочках **(pipelines)** с piping `u7c` для комбинирования.

Основные фильтры:


wc # (word count), Подсчёт строк, слов, символов.

sort # Сортировка строк.

uniq # Удаление дубликатов (только соседних)

head # Вывод первых N строк.

tail # Вывод последних N строк.

[cut # Вырезка полей из строк (по разделителям).](cut)

paste # Слияние строк из файлов по колонкам.

tr # Замена или удаление символов.

sed # Потоковый редактор для поиска\замены (более продвинутый, но вводный уровень).
