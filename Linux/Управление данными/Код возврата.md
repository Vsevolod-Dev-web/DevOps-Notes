`$?` - выводит код возврата последней команды.

```bash
ls /home
echo $?

>> 0
```

Код возврата - это система ядра Linux которая позволяет категоризировать ошибки. 
Всё просто:
- Код возврата 0 - хорошо, всё работает;
- Код возврата 1-255 - плохо, что-то сломалось;

Но стоит быть внимательным. См. Practice Notes/test.sh. `$?` возвращает именно код возврата последней команды.

```bash
#!/bin/bash

deploy --check >> check.log
backCode=$?
if [ $backCode -eq "0" ]
then
    echo "OK"
elif [ $backCode -eq "1" ]
then
    echo "Do not have permission"
elif [ $backCode -eq "2" ]
then
    echo "Do not find config"
fi
```

Если бы я не записал код возврата в переменную `backCode=$?` и вызывал бы его в каждом условии [if/elif](Условия), то он выводил бы код if, elif и т.д.
