Форматирование - это создание [файловой системы](Файловая%20система) на разделе. Буквально, создание формы. После форматирования разделу присваевается [UUID](Постоянное%20монтирование).

Команда для форматирования `mkfs` (make filesystem).

```bash
sudo mkfs.ext4 /dev/sdb1
```
Или,
```bash
sudo mkfs -t ext4 /dev/sdb1
```
