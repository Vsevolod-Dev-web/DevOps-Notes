Подключил диск -> выделил раздел -> форматировал в ФС -> смонтировал -> добавить запись в fstab

Пример:

1. `sudo fdisk /dev/sdb` — создать раздел
2. `sudo mkfs.ext4 /dev/sdb1` — отформатировать
3. `sudo blkid /dev/sdb1` — узнать UUID
4. `sudo nano /etc/fstab` → `UUID=... /mnt/data ext4 defaults 0 2`
5. `sudo mount -a` — применить без перезагрузки
