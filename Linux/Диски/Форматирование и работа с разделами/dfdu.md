`df / du` две утилиты в Linux позволяющие отслеживать использование дискового пространства.

`df` - позволяет отслеживать полное\занятое\свободное место на дисках
`du` - позволяет увидеть сколько дискового пространства занимают отдельные файлы.

```bash
# Как правило, обе эти команды использутся с параметром -h для более
# Человеко-читаемого отображения

df -h
du -h
```

У утилиты `du` есть флаг позволяющий определить глубину поиска. `-d` он же `--max-depth <N>`. Этот флаг критичен для использования, директория может быть очень большой, тогда для её сканирования потребуется очень много времени.А сканировать каждый файл не всегда нужно. Для общей диагностики занятости место достаточно увидеть вес верхних директорий

###  Базовый сценарий использования
1. Приходит алёрт с сервера. Диск заполнен на 95%.
2. Смотрим заполненность нашего диска командой `df -h`
3. Предположим, что у нас главный раздел который исп. 90% памяти диска - /home. Смотрим, он заполнен на 90%.
4. Сканируем этот раздел командой `du -hd 1 /dir-name | sort -h` при этом, сортируем по AZ с флагом человеко-читаемости.
5. Смотрим какая директория или какой файл занимает больше всего места. 
6. Если это директория - возвращаемся на пункт 4
