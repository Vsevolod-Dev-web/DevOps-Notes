Команда `read` - встроенная команда bash, которая читает строку [stdin](Дескриптор) и сохраняет его в переменную.

Команда просто ждёт ввода от пользователя и кладёт в [переменную](Переменные) `name`.
```bash
read name
```

Можно использовать с подсказкой, параметр `-p`:
```bash
read -p "How is your name? " name
```
