Конструкция для выполнения блока кода несколько раз, или для итеративного прохода по множествам.

#### for циклы.
Итерация по списку элементов (файлы, аргументы, диапазоны).

Конструкция
```bash
for var in [list]
do
	commands
done
```

Пример
```bash
for planet in "Mercury Venus Earth Mars Jupiter"
# Все планеты в одной линии
# 
do
	echo $planet
done
```

#### While-циклы
Выполнение блока кода итеративно, до тех пор пока ИСТИННО условие.

Конструкция:
```bash
while [ condition ]
# Spaces, because these are "test-brakets"
do
	command(s)
done
```

Пример: 
```bash
var0=0
LIMIT=10
while [ "$var" -lt "$LIMIT" ]
do
	echo -n "$var0 " # -n suppresses newline
	var0='expr $var0 + 1' # var0=$(($var0+1)) also works
done
```

#### Until-циклы
Инверсия While, выполняет блок кода итеративно пока условие ложно.

Конструкция:
```bash
until [ condition-is-true ]
do
	command(s)
done
```

Пример:
```bash
END_CONTIDITON=end

until [ "$var1" = "$END_CONDITION" ]
do
	echo "Input variable #1 "
	echo "($END_CONDITION to exit)"
	read var1
	echo "variable #1 = $var1"
	echo
done
```
