Операторы условия. if/else. Выполняется, ***если*** истинно условие, ***иначе*** выполняется другое.

```bash
a=3
if [ "$a" -gt 0 ]
then
	if [ "$a" -lt 5 ]
	then
		echo "The value of \"а\" lies somehere between 0 and 5."
	fi
fi

# Same as:

if [ "$a" -gt 0 ] && [ "$a" -lt 5 ]
then 
	echo "The value of \"а\" lies somehere between 0 and 5."
fi
```
