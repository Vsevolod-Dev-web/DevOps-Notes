Операторы условий. `if/else/elif`. Как и в любом другом языке - **выполняют блок кода**, если **истинно условие**.

==Синтаксис==
```bash
if [ condition ]
then
	statemant
else
	statemant
fi
```

Для ***операций сравнения*** используются флажки:
- `-gt` — greater than (больше)
- `-lt` — less than (меньше)
- `-eq` — equal (равно)
- `-ne` — not equal (не равно)

Есть еще некоторые флажки для ***проверок данных*** (файлов и строк):
- `-f file` — файл существует?
- `-d dir` — директория существует?
- `-z "$var"` — переменная пустая?
- `=` — строки равны?
