```bash
#!/bin/bash
```

Это строка называется - **shebang**. Фактически, она указывает чем запустить скрипт, когда ты запускаешь его через конструкцию `./script-name.sh`

Без этого, наш скрипт просто текст, даже не [бинарник](Бинарный%20файл).

**Shebang** говорит ядру "используй вот этот интерпритатор для выполнения файла", в нашем случае `/bin/bash`.

Без **shebang** система пытается угадать и чаще всего угадывает, скрипт запускается, но это не надёжно.
