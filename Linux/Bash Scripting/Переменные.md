Переменная - это абстракция для хранения данных (строк, чисел или ссылок). **Переменная** - это именованные ссылки на значения в памяти, ==без строгой типизации==.

Bash трактует все как **строки**, по умолчанию. Но при этом, поддерживает неявное преобразование для арифметики.

#### Объявление
`var="value"` (без пробелов вокруг `=`, value может быть строкой в кавычках или выражением); Иначе, если попытаться объявить переменную так: `var = "value"`, то система воспримет как вызов команды `var` с аргументами `=` и `"value"`
#### Передача данных в переменную
Для передачи **данных из команды**, используется синтаксис: 
```bash
var=$()
```

Пример:
```bash
#!/bin/bash
linesCount=$(wc -l scores.txt)
```
Сперва вычисляется то, что в скобках, после чего результат записывается в переменную.

Для **арифметических выражений**, используется синтаксис: 
```bash
var=$(())
```

Пример:
```bash
sum=$((2 + 3))
```
Сперва вычисляется выражение в двойных скобках, после чего, результат записывается в переменную.

#### Доступ
`$var` или `${var}` (для безопасного использования в строках или с модификаторами)

**Пример**:

```bash
name="Vsevolod"
echo "Hello, $name"
```
##### Параметр по умолчанию
В переменную можно записать дефолтное значение. Для этого используется синтаксис:
```bash
${name:-"Anonim"}
```

Для примера:
```bash
#!/bin/bash
read -p "How is your name? -" name
currDate=$(date)

echo "Hello, ${name:-"Anonim"}! Today: $currDate"
```
Стоит понимать, что стандартное значение для переменной присваивается не при её объявлении или инициализации, а при её вызове.
