Переменная - это абстракция для хранения данных (строк, чисел или ссылок). **Переменная** - это именованные ссылки на значения в памяти, ==без строгой типизации==.

Bash трактует все как **строки**, по умолчанию. Но при этом, поддерживает неявное преобразование для арифметики.

**Объявление**: `var="value"` (без пробелов вокруг `=`, value может быть строкой в кавычках или выражением);

**Объявление с параметрами по умолчанию:** `${var_name:-default_value]`

**Доступ**:  `$var` или `${var}` (для безопасного использования в строках или с модификаторами)

**Пример**:

```bash
name="Vsevolod"
echo "Hello, $name"
```

Для передачи данных из команды в переменную используется синтаксис `$()`

Пример:
```bash
#!/bin/bash
linesCount=$(wc -l scores.txt)
```

Для целочисленного сложения двух значений используется синтаксис `$(())`

Пример:
```bash
sum=$((2 + 3))
```
