**Порт** - это числовой идентификатор на уровне одного конечного устройства на который можно направлять данные из сети.

*Аналогия. ip-адрес - это многоквартирный дом. Порт - это номера квартир в этом доме*

ip-адрес идентифицирует устройство в сети. Порт - определяет определенный сервис на этом устройстве куда можно направить данные.

Порт можно "открыть" (в Linux чаще означает **разрешить в firewall**) - это значит назначить порт к определенному сервису. Когда порт открывается на определенном сервисе, значит он будет постоянно обрабатывать данные входящие на этот порт. 

### Стандартные порты
Порты которые по стандарту зарезервированы за определенными сервисами:

| Порт | Сервис                               |
| ---- | ------------------------------------ |
| 22   | SSH                                  |
| 80   | HTTP                                 |
| 443  | HTTPS                                |
| 5432 | PostgreSQL                           |
| 3306 | MySQL                                |
| 6379 | Redis                                |
| 8080 | HTTP альтернативный (для разработки) |
