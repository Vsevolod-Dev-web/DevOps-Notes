**Default gateway** - это узел, как правило роутер, суть узла - быть "выходом" для сетевого трафика из локальной сети во внешние сети.

По сути, когда компьютер не знает куда направить трафик, по стандарту он направляет его на роутер, он уже разберётся. Это и есть Default Gateway.

`ip route` - показывают таблицу маршрутизации, где и указан default gateway. 
```bash
$ ip route
default via 192.168.5.1 dev wlp1s0
```
192.168.5.1 - адрес роутера.

Так же, в таблице маршрутизации можно увидеть следующую строку: 
```bash
$ ip route
192.168.5.0/24 dev wlp1s0
```
Если нужно отправить что-то в сеть, которая начинается на 192.168.5.x - то эта запись говорит компьютеру, что это текущая сеть. Не нужно передавать это на роутер, разберёмся тут. ==Так трафик не входит во вне, не идёт на **gateway**.==
