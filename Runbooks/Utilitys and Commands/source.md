```Shell
source
```

**Source** - это встроенная команда, которая читает и исполняет содержимое файла в текущем shell-окружении. Альтернатива запуска подоболочки (subshell).

Это позволяет применить определенные изменения (установить переменные, определить функцию или сменить директорию) на текущую сессию в терминале. 

Вот некоторые примеры:

1. Активация [виртуального окружения ](Python%20venv.md)python:

```bash
source venv/bin/activate
```

2. Применение настроек в bash: 
	Если вы изменили файл .bashsrc или .bash_profile, используйте команду, чтобы не перезапускать терминал:
```bash
source ~/.bashrc
```

3. Запуск скрипта в текущей оболочке:
	В отличие от `./script.sh`, команда `source script.sh` выполнит скрипт без создания дочернего процесса, что позволяет изменять переменные текущей сессии.

4. **Загрузка переменных окружения из файла:**
	Если у вас есть файл `config.env` с содержимым `DB_HOST=localhost`, команда `source config.env` сделает переменную `$DB_HOST` доступной в текущей сессии.
