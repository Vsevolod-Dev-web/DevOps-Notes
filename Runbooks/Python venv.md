Иногда, необходимо изолированно поставить пакеты для некоторых сервисов. Скажем, на одной системе работают два приложения - одно использует pip3, а другое использует pip4. Если установить пакеты на систему:

```bash
sudo apt install pip4
```

то, pip4 перерешит пакеты pip3 и приложение на последнем сломается. Для этого, конкретно в python есть модуль **VENV** (Virtual Environment) - изолированное окружение. Его суть - создать изолированное окружение для избежания **[Dependency Hell](Зависимости)**
