SELECT - оператор, позволяющий получать данные из базы. Вся работа с SQL заключается в том, что используя такие операторы мы определяем какие данные и как мы хотим их получить, а так же как мы хотим их представить перед возвратом.

Таблица - абстракция. Строки - экземпляры этой абстракции. Столбцы (поля) же в свою очередь будут представлять общие свойства, характеристики абстракции.

# 1. SELECT, FROM, WHERE
80% запросов в SQL имеют следующую структуру:
```sql
SELECT [что]
FROM [откуда]
WHERE [условие]
```
