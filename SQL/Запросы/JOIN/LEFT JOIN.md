Присоединение "левой" (та, что указывается после FROM) таблицы к "правой" (та, что указывается после JOIN), с учётом совпадений.

## LEFT JOIN
`LEFT JOIN` - все из левой таблицы, даже без пары:

```sql
SELECT c.name, l.amount
FROM clients c
LEFT JOIN loans l ON c.id = l.client_id;
```

Результат:

| name  | amount |
| ----- | ------ |
| Иван  | 5000   |
| Иван  | 3000   |
| Мария | 8000   |
| Пётр  | NULL   |

Пётр есть, но `amount` = NULL — займов нет, значит нечего подставить.
